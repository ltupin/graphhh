Arborescence
```shell
/
├── app.py
├── templates/
│   └── index.html
├── static/
│   ├── css/
│   │   └── styles.css
│   └── js/
│       └── graph.js
├── data/
│   └── graph_data.json
└── requirements.txt
```

Commands
```shell
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
python3 app.py
```

Open browser and navigate to http://127.0.0.1:5000.
